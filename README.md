# Where's my beer?

## Gameplay
* 2D platfomer
* Randomly spawning beers
* Beer-o-meter
* When he collects 6 cold beers and fills up his beer-o-meter
* Warm beers drop down his beer-o-meter
* If the beer-o-meter reaches 0 it's game over
* If the beer-o-meter fills up you win

## Character movement
* Holds beer case
* Walks
